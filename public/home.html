About


The problem space is how to make learning a new language easier for busy or beginner users to conveniently learn and engage with the Somali language. My solution for this pain point is to create SomaliVocals, a Somali language learning app designed to guide users to engage with the phonology of the language. The app uses prominent phonological characteristics that form the foundation of the Somali language, so the user will be able to pronounce and learn the alphabets and words easily.
